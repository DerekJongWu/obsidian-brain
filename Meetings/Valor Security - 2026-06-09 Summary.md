## Product: Reverse-Proxy Security Layer for Agentic AI Apps

- **Core insight:** Vertical SaaS AI apps (Sierra, Vesta, Dekagon, Wonderful) are publicly exposed to the internet AND have deep access to enterprise data.
- **Existing tools (Zscaler, Netskope, Witness AI)** protect inside-out traffic; **Valor protects outside-in.**
- **Positioned as the modern CASB for agentic workflows:**
  - Input control
  - Prompt inspection
  - File scanning
  - Context isolation engine — limits the context the caller gets
- **Key differentiator vs. Witness AI:**
  - Witness monitors **employee use of AI tools inside the org** (identity-aware, inside-out)
  - Valor secures the **business process itself**, not the identity or internal user (outside-in)
