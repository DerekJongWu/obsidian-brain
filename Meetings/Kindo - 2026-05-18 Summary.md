### Product & Technology

- **General-purpose AI orchestration engine for security teams**
  - Runs fully on-prem, self-managed cloud, or SaaS
  - **Universal chatbot + no-code agent builder** (shipped September)
  - **Autonomous agents** running 24/7 on triggers/schedules
- Core innovation: **Linux sandbox (Kali Linux default)**
  - Agents can write/compile code, access databases, network files, run security tools
  - **100+ MCP servers** for security tool integration
- **Single pane of glass across entire security stack**
  - Can work alongside vertical solutions but eliminates need to log into separate tools
  - Customers find they stop using individual tools and work natively in Kindo
- **Enterprise governance layer**
  - Role-based access controls, audit trails, secrets management
  - Data loss prevention integration
  - Model-agnostic (OpenAI, Chinese models, homegrown models)
