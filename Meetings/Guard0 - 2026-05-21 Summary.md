## Company Overview & Product

- Guard0 provides AI agent security across three layers:
  - **Data/analytics agents** (Databricks, Snowflake) — controlled environments
  - **IT companion agents** (platform-based) — broader access
  - **Procured agents** (Workday, ServiceNow, Zoom) — vendor solutions
- Three core product capabilities:
  1. **Posture** — visibility via integrations (95% surface coverage), no eBPF/proxy friction
  2. **Red teaming** — customized test cases generated for each environment using SLMs
  3. **Runtime enforcement** — behavior evaluation against intent/governance policies
- Recent detection example: agent accessing restricted SaaS data, dumping to Excel accessible by unauthorized users
- Platform runs entirely on-premises with custom SLMs
- Self-serviceable onboarding without customer-success involvement
- Cost: <$5K/month for an 80–90K employee organization
