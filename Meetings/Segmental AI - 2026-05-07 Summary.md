## Product & Technical Approach

- Knowledge-graph platform for financial-services workflows
- Core value proposition: automate entire workflows (vs. selling tools to operators)
  - Extract, classify, normalize unstructured data (PDFs, emails, Excel)
  - Build purpose-built knowledge graphs per workflow
  - Layer agents for autonomous execution with minimal human-in-the-loop
- Key differentiator: context graph + agents (vs. agents alone)
  - 95–98% of queries resolved by the deterministic knowledge graph (not LLM)
  - Significantly lower token costs vs. vanilla agentic competitors
- Deployment: on-premise within customer firewall (AWS, GCP, Azure)
