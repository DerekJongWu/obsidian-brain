## Company Overview & Background

- Diversion: development platform and version control system for scale
  - Founded August 2022 by [[Sasha Medvedovsky]] (CS master's, MBA, 2 previous exits)
  - Previous exits: algo-trading ML platform; early mobile neural networks (~2014–2015)
  - Co-founder from his second startup (Egal Lazarev, CTO)
- Core problem: Git built ~21 years ago, doesn't scale for the AI era
  - Conservative estimate: ~1000x more data creation coming
  - Each engineer running ~10 agents, each ~20x faster than humans
  - GitHub has ~180M users vs. ~30–40M actual developers worldwide
- Target market: large enterprises that couldn't use Git
  - Financial sector, gaming, semiconductors
  - Google/Facebook built their own systems due to Git limitations
  - GitHub CTO cited monorepos as the biggest unsolvable challenge

## Product & Technology

- Cloud-native architecture vs. Git's local-machine constraints
  - Built like Dropbox/Gmail for massive data management
  - Team includes Dropbox alumni (Meir Morgenstern, CSO)
- Current traction: 40K developers signed up, 30K projects, 1M+ commits in production
  - Launched January 2024, spent ~18 months hardening edge cases
  - 500–600 weekly signups, active Discord community (~3K members)
- Migration strategy: synchronization tool with GitHub for gradual transition (releasing in a few weeks)
- Pricing model: $25–50 per seat/month, plus usage-based for AI agents
