## Business Description

- **Problem:** Enterprise software development is dominated by general-purpose AI coding assistants (Copilot, Cursor, etc.) trained on public code, which don't model the customer's specific codebase, conventions, or build pipeline — and which can't safely run autonomously inside the enterprise.
- **Product:** Customizable foundation models for code + software agents that operate within secure, sandboxed environments. Trained with Reinforcement Learning from Code Execution Feedback (RLCEF) — the agent runs code, observes outputs, and learns from execution feedback rather than just static text prediction.
- **Wedge / entry point:** Enterprise contracts where customizable / on-prem-friendly models that integrate with the customer's existing codebase and deploy in a sandboxed environment matter more than the public-internet-coverage of incumbent assistants.

---

## Market & Competition

- **Market size:** Enterprise software-development tooling + foundation-model spend — multi-billion-dollar TAM with GitHub Copilot estimated past $400M ARR; new entrants (Cursor, Anthropic Claude Code, OpenAI Codex) growing fast.
- **Key competitors:** GitHub Copilot, Cursor, Anysphere, Cognition (Devin), Claude Code, OpenAI Codex, Magic, Codeium.
- **Differentiation:** Customizable foundation models + sandboxed agent runtime focused on enterprise deployment — vs. SaaS-first IDE plugins. RLCEF approach (training on code execution outcomes) is technically differentiated from prediction-only LLMs.
