## Business Description

- **Problem:** Existing CRMs (HubSpot, Pipedrive, Zoho, Salesforce) require heavy manual data entry, lack first-class agent execution, and force sales teams to live outside the system of record. Migrations between CRMs are painful enough that incumbents retain by inertia, not value.
- **Product:** AI-native CRM where agents and humans have parity — both can create records, run workflows, join calls, and operate via natural language across the entire CRM surface. Built on a proprietary retrieval system (explicitly *not* a graph database) chosen for scalability.
- **Wedge / entry point:** (1) Greenfield startups (YC, teams running on Notion/spreadsheets) and (2) migrations from incumbent CRMs via a built-in migration agent that ingests CSVs, maps to Lightfield's data model, and deduplicates in ~1 hour.

---

## Market & Competition

- **Market size:** CRM is a >$80B category — Salesforce alone is the incumbent reference comp; AI-native challenger window is wide open given current incumbents' agent retrofit posture.
- **Key competitors:** Salesforce, HubSpot, Pipedrive, Zoho (all incumbents being actively displaced by Lightfield deals); also adjacent AI-CRM challengers (Attio, Clay).
- **Differentiation (as of 2026-05-16):**
  - Human-agent parity — agents are first-class, not bolted on
  - Natural-language interface for *all* CRM operations
  - Proprietary retrieval architecture (not a graph DB) — bet on retrieval scalability over relationship modeling
  - In-product migration agent — collapses switching cost
  - Largest current customer ~50–100 sales reps, so not yet enterprise-validated
